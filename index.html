🚗 App Ubica2

Sistema de gestión de estacionamientos desarrollado con Flask. Incluye autenticación con encriptación de contraseñas, base de datos PostgreSQL, ABM de usuarios, vehículos y parkings, y envío automatizado de correos.

🔐 Sistema de puertas

Proyecto para control de accesos. Dispositivos ESP32 y ESP01 se comunican con un servidor Mosquitto MQTT. El backend, desarrollado con Flask, gestiona llaves digitales, puertas, usuarios y permisos. La base de datos utilizada es MySQL, y el sistema permite registro, autenticación JWT y monitoreo remoto.

♟️ Ajedrez/ Damas

Proyecto académico desarrollado en Java para la materia Diseño de Sistemas (Universidad Austral). Implementa el juego de ajedrez y damas utilizando patrones de diseño y una arquitectura orientada a objetos. Se desarrolló un adaptador para integrar una interfaz gráfica provista por la cátedra, así como toda la lógica de juego, tablero y movimientos válidos.
